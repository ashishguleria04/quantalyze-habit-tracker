# Quantalyze

A minimalistic habit tracker designed to optimize vitality, focus, discipline, and social connection.

## Tech Stack

- **Framework**: Next.js 15
- **Database**: Supabase
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## Getting Started

1. **Clone the repository**

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure environment**
   Copy `.env.example` to `.env.local` and add your Supabase credentials:
   ```bash
   cp .env.example .env.local
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

## Features

- **Habit Management**: Create and track daily habits with custom weights and goals.
- **Analytics**: Visualize progress through quality scores and consistency metrics.
- **Daily Logs**: Quick logging system for tracking performance.
- **Quantified Self**: Insights into Vitality, Focus, Discipline, and Social metrics.
